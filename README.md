## DogFood shop
**DogFood** - Shop selling goods for dogs

## Features 
* User SignUp/SignIn using token
* Personal Area
* Product detail page
* Adding a product to the cart
* Adding a product to favorites
* View/add product reviews
* Sorting goods by category
* Adding a product
* Editing your product
